# BluePilot Remote - PC Connection Steps

For Windows PC connection, do this exact order:

1. On Windows: open Bluetooth settings.
2. Remove any old entries named BluePilot, Android phone, or old phone pairing.
3. On phone: open BluePilot Remote.
4. Tap **Prepare PC connection**.
5. Accept Android discoverability dialog.
6. On Windows: click **Add device > Bluetooth**.
7. Select this phone / BluePilot Remote.
8. Accept the pairing code/dialog on both PC and phone.
9. Wait for BluePilot status to show **Connected**.
10. Test mouse/keyboard.

Important notes:
- The phone must support Android Bluetooth HID Device mode. Some phones/ROMs block it.
- If it pairs but does not connect, remove the pairing from Windows and phone, then repeat.
- Do not connect from only Android Bluetooth settings first; prepare BluePilot host mode first.
